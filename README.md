# Next Project

一个基于 Next.js App Router 的基础项目骨架，默认使用 TypeScript。

## 启动

```bash
npm install
npm run dev
```

默认访问地址：

```bash
http://localhost:3000
```

## 目录结构

```bash
src/
  app/            # 路由与页面
  components/     # 可复用组件
  lib/            # 工具函数与公共逻辑
  styles/         # 全局样式
public/           # 静态资源
```
